# Changelog

Все значимые изменения в проекте ChatList будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.1] - 2024-XX-XX

### Добавлено
- Диалог ввода API-ключа с возможностью сохранения в .env файл
- Автоматическое создание папки данных в пользовательской директории
- Поддержка хранения файлов данных в %LOCALAPPDATA%\ChatList
- Улучшенная обработка отсутствующих API-ключей

### Исправлено
- Ошибка прав доступа при установке в Program Files
- Проблемы с созданием файлов логов и базы данных после установки
- Поиск .env файла в правильных директориях

### Изменено
- Файлы данных (БД, логи, .env) теперь хранятся в пользовательской папке
- Улучшен интерфейс диалога ввода API-ключа

## [1.0.0] - 2024-XX-XX

### Добавлено
- Первый публичный релиз
- Отправка промтов в несколько нейросетей одновременно
- Сравнение ответов в табличном виде
- Сохранение промтов и результатов в базу данных SQLite
- Управление моделями (добавление, редактирование, удаление)
- Поиск по сохраненным промтам
- Экспорт результатов в Markdown и JSON
- Поддержка OpenAI, DeepSeek, Groq, OpenRouter API
- Улучшение промтов с помощью AI-ассистента
- Настройка темы и размера шрифта
- Инсталлятор для Windows (Inno Setup)

---

## Типы изменений

- **Добавлено** - для новых функций
- **Изменено** - для изменений в существующей функциональности
- **Устарело** - для функций, которые скоро будут удалены
- **Удалено** - для удаленных функций
- **Исправлено** - для исправления ошибок
- **Безопасность** - в случае уязвимостей
